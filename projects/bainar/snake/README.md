# Micro:bit Snake

## Author

David Bainar

## Description

A simple Snake game for the BBC micro:bit.

Control the snake, eat apples, and grow as long as possible without
crashing into yourself.

The game features three difficulty levels with increasing speed.

## Controls

- **Button A** — turn snake left
- **Button B** — turn snake right

## Scoring and Levels

Each apple increases the snake’s length.

When the snake reaches a length of 20, the next level begins and the
game speed increases.

The game contains three levels of increasing difficulty.
