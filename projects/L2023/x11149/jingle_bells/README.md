# Micro:bit projekt

## Autor
Ondřej Sotolář
## Popis
Po přejetí prstu přes logo Micro:bitu se na displeji zobrazí hvězdička a současně začne přehrávat melodie písně Jingle Bells. Hvězdička zůstává zobrazena po celou dobu přehrávání melodie. Po jejím skončení se na displeji zobrazí text: „Vánoce jsou za rohem.“
## Ovládání
Ovládání je jednoduché, stačí jen přejet prstem po logu a micro:bit už dělá vše sám
