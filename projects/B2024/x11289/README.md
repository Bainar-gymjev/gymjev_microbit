# Micro Space Dodger

A simple arcade game made with Micro and Python.

## Author

Matěj Španiel

## Description

This project is a simple space game created using a BBC Micro and Python.

The player controls a rocket at the bottom of the screen and must avoid falling meteors. As the game progresses, the meteors fall faster, making the game more challenging.

The game ends when a meteor hits the rocket. The final score is then displayed on the Micro screen.

## Controls

Button A → Move the rocket left
Button B → Move the rocket right
Buttons A + B → Restart the game after losing
How It Works
The rocket starts in the middle of the bottom row.
Meteors randomly appear at the top of the screen.
The player moves the rocket to avoid incoming meteors.
Each meteor that passes the rocket increases the score.
The game speed gradually increases as the score grows.
If a meteor collides with the rocket, the game ends and the score is displayed.
Press A + B to start a new game.

## Requirements

BBC Micro
MicroPython

## Features

Randomly generated meteors
Increasing difficulty
Score tracking
Game restart option
Simple and responsive controls
