# Microbit projekt

## Autor:
Sebastian Roche
# Popis:
Po zaznamenání tlesknutí micro:bit postupně zobrazí šipky otáčející se kolem dokola. Následně se objeví symbol Pac-Mana a spustí se úvodní herní melodie. Po jejím dohrání se zobrazí smajlík vyjadřující radost a číslo 67.
## Ovladání:
Celý program se ovládá pomocí tlesknutí rukou nebo jiného hlasitého zvuku. Po zaznamenání hlasitého zvuku micro:bit zobrazí šipky otáčející se dokola, následně se objeví Pac-Man a přehraje se úvodní herní melodie. Po jejím dohrání se zobrazí usměvavý smajlík a nakonec číslo 67.
